# trap

A terminal UI (TUI) app that combines productivity tools with a radio player — all in one place.

## Features

- **Daily To-Do List** — manage your daily tasks without leaving the terminal
- **Radio Player** — stream internet radio stations directly from the TUI, powered by [mpv](https://mpv.io/)

## Prerequisites

- **Go** 1.25.8+
- **mpv** — must be installed and available in your `PATH` for radio streaming
